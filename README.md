# 浪浪记账

## 简介

浪浪记账是一款简单又好看的记账应用，也是一款基于 Vue、VueRouter、Vuex、TypeScript 的单页面应用

## 知识

遇到了很多零零散散的知识，在此汇集一下

* Vue Router
* 封装组件和应用组件
* 使用插槽
* 引入 svg
* 数据传递
* 全局数据管理
* HTML、CSS、SCSS
* JavaScript + TypeScript
* Vue 装饰器
* 命名习惯
* mixins
* echarts

## 一些 bug

* windows.alert、windows.prompt、windows.confirm 在有些浏览器很霸道，体验不是特别好
